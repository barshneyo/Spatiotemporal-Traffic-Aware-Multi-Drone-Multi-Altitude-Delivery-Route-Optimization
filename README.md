# Spatiotemporal-Traffic-Aware-Multi-Drone-Multi-Altitude-Delivery-Route-Optimization
A clustering-driven delivery allocation framework using Mini-Batch K-Means, DBSCAN, OPTICS, and a custom Congestion-Guided Residual Graph Clustering (CGRGC) method to partition dynamic demand regions integrated with a time-expanded graph routing framework using GNNs for congestion-aware state representation and RL for adaptive decision-making, benchmarking Prioritized A, MCTS, DHC, and SHC for multi-drone route optimization under altitude constraints.
